# About

- This project mainly acts as a connection between me and other people. It's is primarily built using vite for fast development and optimized builds.

# First steps

- Run these prompts on the root directory
  -npm install (Download necessary node modules)
  -npm run build (Will optimize and bundle source code into a dist folder)

# Overview

- To run the app, use
  -npm run dev (Will show the local host link to view the app locally)
